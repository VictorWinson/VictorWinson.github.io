system installing#

  1. Ubuntu 20.04LTS (place in the living room used both as a media player for my parents and a 24*7 was)

  2. OMV (when I found my parents seldom use it, I change into OMV for more convenient raid experience)

  3. FreeNAS (freeBSD need VM to run docker, when you are using a i3 processor it really drives people crazy)

  4. OMV (yes, I’m back)

  5. TrueNAS scale (useable ZFS+ Debian + docker, this is the dream NAS OS)

service running/been ran on it#

  1. Nextcloud (appreciate by my mum so she don’t need to use WeChat or flash drive to keep her documents synchronised)

  2. Jellyfin (use aria2 to download movie and Jellyfin to manage them is a good choice)

  3. No-ip (DDNS + Dynamic external IP is a solution to access it when you are at school, though expose you NAS can be danger for you data, so make sure you know what you are doing)

  4. PhotoPrime (google open-source gallery, use tenser flow to classify photos, after using iCloud this had been abandoned)

  5. Aria2 pro (best aria2 docker ever found, move when complete, auto update tracker-list and auto delete temp file when complete)
