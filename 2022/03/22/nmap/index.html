nmap

what is nmap#

conclusion of what can nmap does:

  • host discovery
  • port discovery/ enumeration
  • service discovery
  • OS version detection
  • MAC address detection
  • service version detection (eg. apache nginx mysql)
  • vulnerability/ exploit detection, using namp scripts
  • nmap IDS/ portscan deteciton

simple commands#

namp -sP 192.168.0.0/16#

/16 scan from 192.168.0.1 - 192.168.254.254
-sP stands for ping scan

namp -p 1-1000 -Pn -sV -sS -T4 target#

-p 1-1000 specify port 1-1000
-sV service version deteciton
T4 scaning speed(1-5), larger number usully fast but could lead to inaccurate.
-sS SYN scan
Pn treat all host online

nmap -v -sS -A -T4 target#

-v verbose output
-A ALL usually quite slow (include OS and version detection + traceroute and scripts against target services)

namp -O -sV -sS -T5 target#

-O OS and version detection

nmap -iL ip-address.txt#

scan IPs from files

nmap -p- target#

nmap scan all ports

nmap -oG grep-output.txt target#

oG grepable output

nmap target --webxml -oX - | xsltproc --output file.html -#

output a HTML report
